## Database

``` sql
CREATE DATABASE employee_portal;

USE employee_portal;

CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100),
    email VARCHAR(100) UNIQUE,
    password VARCHAR(100),
    phone VARCHAR(20),
    salary DOUBLE,
    designation VARCHAR(100)
);
```

## Architecture

MVC Pattern: - **Model:** User.java - **View:** HTML pages + HTML
generated by servlets - **Controller:** Servlets - **DAO:** UserDAO -
**Database:** MySQL

## Flow

1.  Home (`index.html`)
2.  Register (`RegisterServlet` → `UserDAO.register()`)
3.  Login (`LoginServlet` → `UserDAO.login()`)
4.  Create session after successful login.
5.  Profile loads employee data using `getUserByEmail()`.
6.  Edit updates name, email and phone.
7.  Delete removes the account.
8.  Logout invalidates the session.
